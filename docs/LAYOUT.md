## 1. 全站通用组件(Common Atoms)

### 1.1 顶部导航(Top Bar)

```
+================================================================+
|  AGENT-MAIL // {SECTION}     [ THEME ▾ ]  [ SIGN IN ]  [ > ]   |
+================================================================+
```

- 左:`AGENT-MAIL // {SECTION}` — `//` 作为分隔符,section 全大写
- 中(右):`[ THEME ▾ ]` 主题切换器(见 §3.14),始终可见
- 右:操作按钮(直角黑底白字),未登录显示 `SIGN IN`,登录后显示 `LOGOUT` + 当前邮箱

### 1.2 通用按钮(Button)

```
[ > ACTION LABEL ]       -- 主按钮(黑底白字)
[ CANCEL ]               -- 次按钮(米底黑字,1px 黑边)
[ > DELETE ]             -- 危险按钮(琥珀底黑字)
```

- Hover 时主按钮反转为「黑字绿底」
- Loading 时变为 `[ ... ACTION ]`

### 1.3 输入框(Input)

```
LABEL         : [_____________________________________]
                > 辅助说明文字
```

- 默认仅底部 1px 黑边,聚焦时变为完整 1px 边框
- 左侧保留 `>` 提示符位置(`> [____]`)

### 1.4 卡片 / 模块(Card · Header Strip)

```
+======================================================+
|  HEADER STRIP // TITLE                                |   <-- 黑底白字标题栏
+======================================================+
|  CONTENT AREA                                         |   <-- 米底黑字内容
|  > ...                                                |
+======================================================+
```

### 1.5 状态标签(Status Chip)

```
( ACTIVE ) / ( VERIFIED ) / ( OFFLINE ) / ( WARNING )
```

### 1.6 列表行(List Row · Index)

```
[ 01 ] item-name ....................... meta  [ > ]
[ 02 ] item-name ....................... meta  [ > ]
```

### 3.6 `/dashboard/apikey` — API Key 管理

```
+================================================================+
|  AGENT-MAIL // API KEY    alice@agent.qq.com  [ THEME ▾ ]  [ LOGOUT ]  |
+================================================================+
|                                                                |
|  ## API KEY MANAGEMENT                                         |
|  ----------------------------------------------------------    |
|  > 你的 API Key 用于 CC 启动时调用云端 API。                   |
|  > 每个账号最多持有一个 Key。                                  |
|  > 创建 / 重新生成后,新 Key 会同步到本地 Agent.md。            |
|                                                                |
|  ## CURRENT KEY                                                |
|  ----------------------------------------------------------    |
|  STATUS      : ( ACTIVE )                                      |
|  CREATED AT  : 2026-06-01 12:00 (UTC+8)                       |
|  LAST USED   : 2026-06-27 09:28                                |
|                                                                |
|  KEY VALUE                                                     |
|  +------------------------------------------------------+     |
|  |  amk_a7f9b2c4d1e8f6a3b5c2d9e1f4a7b8c9d2e5f8a1b4c7  |     |
|  +------------------------------------------------------+     |
|              [ COPY ]  [ REGENERATE ]  [ DESTROY ]            |
|                                                                |
|  > ( WARNING ) REGENERATE 会立即作废旧 Key。                   |
|  > ( WARNING ) DESTROY 后 CC 将无法调用云端 API。              |
|                                                                |
|  ---                                                           |
|                                                                |
|  ## NO KEY STATE(若尚未创建)                                   |
|  ----------------------------------------------------------    |
|  STATUS      : ( NONE )                                        |
|                                                                |
|  > 你还没有创建 API Key。                                      |
|  [ > CREATE API KEY ]                                          |
|                                                                |
+================================================================+
```

**关键交互细节**
- `COPY` 点击后状态行变为 `( COPIED )` 并保留 3 秒后回到 `( ACTIVE )`
- `DESTROY` 需要二次确认弹窗
- `REGENERATE` 二次确认 + 旧 Key 立即失效
